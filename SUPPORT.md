# Support

## How to Get Help

### Found an Incorrect Answer?

Open an issue using the [Bug Report](https://github.com/muhittincamdali/mobile-interview-questions/issues/new?template=bug_report.yml) template.

### Want to Request New Questions?

Open an issue using the [Feature Request](https://github.com/muhittincamdali/mobile-interview-questions/issues/new?template=feature_request.yml) template.

### General Discussion

Use [GitHub Discussions](https://github.com/muhittincamdali/mobile-interview-questions/discussions) for:

- Interview tips and strategies
- Question clarifications
- Study group coordination
- Career advice

### Contributing

See [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines on how to add questions or improve answers.
