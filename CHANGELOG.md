## [1.0.0] - 2026-01-15

### Added

- 300+ iOS interview questions covering Swift, UIKit, SwiftUI, concurrency, architecture, and system design
- 300+ Flutter interview questions covering Dart, widgets, state management, platform channels, and testing
- 200+ React Native interview questions covering JavaScript/TypeScript, hooks, navigation, and native modules
- 200+ General mobile development questions covering performance, security, CI/CD, and behavioral topics
- Detailed answers with code examples for every question
- Difficulty ratings (Junior, Mid, Senior) for all questions
- Dedicated deep-dive files for Swift fundamentals, system design, Dart fundamentals, and JavaScript fundamentals
- Turkish summary (README_TR.md)
- Contributing guidelines and issue templates
- CI pipeline with markdown linting
